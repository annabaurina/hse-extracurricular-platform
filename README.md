# Веб-платформа для студенческих внеучебных организаций

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-2.3.3-green)](https://flask.palletsprojects.com/)
[![SQLAlchemy](https://img.shields.io/badge/SQLAlchemy-ORM-brightgreen)](https://www.sqlalchemy.org/)

## Основные возможности

### Аутентификация и пользователи
- **Регистрация с подтверждением email** через верификационные коды
- **Ролевая модель**: root_admin, admin, user
- **Управление профилем**: смена email, пароля, личных данных

### Организации и управление
- Создание и редактирование студенческих организаций
- **Категории организаций** (управление root_admin)
- Назначение лидеров организаций
- Управление участниками организаций

### Вакансии и отклики
- Публикация вакансий от имени организаций
- **Система откликов** с заявками от студентов
- **Поиск по вакансиям**
- Управление статусами заявок (принято/отклонено/рассматривается)
- Автоматическое добавление принятых кандидатов в организацию

### Мероприятия и регистрация
- Создание мероприятий с датой и описанием
- **Система регистрации** на мероприятия
- Просмотр списка зарегистрированных участников
- Календарь предстоящих событий

### Административная панель
- **Полный контроль** над пользователями, организациями, вакансиями, мероприятиями
- Назначение прав администратора
- Создание организаций и назначение лидеров
- Управление системными настройками

## Технологический стек

### Backend
- **Python 3.8+** - основной язык программирования
- **Flask** - веб-фреймворк
- **SQLAlchemy** - ORM для работы с базой данных
- **Flask-Login** - аутентификация и сессии
- **Flask-Mail** - отправка email уведомлений
- **Werkzeug** - хеширование паролей

### Frontend
- **HTML/CSS** - базовая верстка
- **Jinja2** - шаблонизация
- **Bootstrap** (предположительно) - стилизация

### База данных
- **SQLite** - реляционная база данных
- **SQLAlchemy ORM** 

### Предварительные требования
- Python 3.8 или новее
- pip (менеджер пакетов Python)
- Почтовый сервер (для отправки верификационных кодов)
